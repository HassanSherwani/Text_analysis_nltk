- Stopwords

- Lemmas, Synonyms and Antonyms
- Wu Palmer Similarity
- Path and LCH(*Leacock Chodorow) Similarities
- Bigrams & Trigrams
- Stemming
- Lemmatization
- Regular Expression
- Replacer
